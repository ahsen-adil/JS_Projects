# Todo List Application

This is a simple, client-side Todo List application built with HTML, CSS, and JavaScript. It allows users to add, edit, and delete todo items directly in their browser.

## Features

*   **Add Todo Items**: Easily add new tasks to your list.
*   **Edit Todo Items**: Modify existing tasks using a prompt.
*   **Delete Todo Items**: Remove tasks you no longer need.
*   **Responsive Design**: Basic styling for a clean user interface.

## Technologies Used

*   **HTML5**: For the basic structure of the web page.
*   **CSS3**: For styling and layout.
*   **JavaScript**: For all interactive functionality.

## Installation

1.  **Clone the repository (if applicable) or download the files:**
    If this project is part of a larger repository, you would typically clone it using `git clone [repository_url]`.
    Otherwise, simply download the `index.html`, `style.css`, and `app.js` files.

2.  **Open `index.html`**:
    Navigate to the directory where you saved the files and open `index.html` in your preferred web browser.

## Usage

1.  **Add a Todo**: Type your task into the input field and click the "Add Todo" button.
2.  **Edit a Todo**: Click the "Edit" button next to the todo item you wish to modify. A prompt will appear allowing you to change the text.
3.  **Delete a Todo**: Click the "Delete" button next to the todo item you wish to remove.
